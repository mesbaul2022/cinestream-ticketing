# Sprint 1 Retrospective

**Sprint goal:** Repo, skeleton apps, and basic auth working end-to-end.
**Result:** All 7 tasks completed (Project Setup + User Management).

## What went well
- Branch-per-task + PR review flow worked smoothly once we got the hang of it.
- Registration and login tested successfully end-to-end.

## What was confusing
- PR base branch defaulted to `main` instead of `develop`, which briefly caused `develop` and `main` to go out of sync.
- Jira's "Backlog" vs "Board" distinction took a moment to click.

## What we'll change next sprint
- Double-check the base branch dropdown on every PR before creating it.
- Delete feature branches immediately after merging, instead of batching cleanup later.
