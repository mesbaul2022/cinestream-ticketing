## Tech Stack
HTML, CSS, PHP, MySQL (XAMPP)

## Setup
1. Install XAMPP, start Apache + MySQL.
2. Clone this repo into `htdocs`.
3. Import `database/cinestream.sql` via phpMyAdmin.
4. Visit `http://localhost/cinestream-ticketing/`.

## Features
- User registration/login
- Movie browsing & showtimes
- Seat selection & booking confirmation
- Admin panel for movies, showtimes, and bookings
